## Краткое описание

Система мониторинга посещаемости организаций дополнительного образования и загруженности обучающимися муниципальных общеобразовательных организаций.

## Программные средства

Языки программирования:

- PHP (Laravel);
- Javascript.

Библиотеки:

- Laravel Excel.
